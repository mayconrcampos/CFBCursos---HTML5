Aula 04 - Tag A - Criando links

O elemento a do SVG cria um hiperlink para outras páginas da web, arquivos, locais na mesma página, endereços de email ou qualquer outra URL. É muito semelhante ao elemento a do HTML.

A tag A é um contêiner, o que significa que você pode criar um link em torno do texto (como em HTML), mas também em torno de qualquer elemento.

Outros valores possíveis para o parâmetro target:

_self

A imagem SVG atual é substituída pelo conteúdo vinculado no mesmo contexto de navegação da imagem SVG atual.

Se omitirmos o target, será adotado valor default _self, abrir na mesma janela.

_parent

O contexto imediato de navegação pai da imagem SVG é substituído pelo conteúdo vinculado, se ele existir e puder ser acessado com segurança a partir deste documento.

_top

O conteúdo da janela ou guia ativa completa é substituído pelo conteúdo vinculado, se ele existir e pode ser acessado com segurança a partir deste documento

_blank

Uma nova janela ou guia sem nome é solicitada para a exibição do conteúdo vinculado, se este documento puder fazê-lo com segurança. Se o agente do usuário não tiver suporte a várias janelas/guias, o resultado será o mesmo que _top.

XML-Name

Especifica o nome do contexto de navegação (guia, quadro inline, objeto, etc.) para exibição do conteúdo vinculado. Se um contexto com esse nome já existir e puder ser acessado com segurança a partir deste documento, ele será reutilizado, substituindo o conteúdo existente. Se não existe, ele é criado (o mesmo que '_blank', exceto que agora tem um nome). O nome deve ser um nome XML válido [XML11], e não deve começar com um sublinhado (caractere U+005F LOW LINE), para atender aos requisitos de um nome de contexto de navegação válido do HTML.
